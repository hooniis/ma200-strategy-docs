## 🎯 기본 원칙 (반드시 먼저 암기)

1. **판단 시점**: 매일 아침 (미국장 마감 후 = 한국시간 아침 6:00~8:00)
2. **판단 기준**: TQQQ **확정 종가** vs TQQQ **200일 이동평균선**
3. **매도는 즉시, 매수는 하루 더 확인**
4. **예측하지 않는다. 신호에만 반응한다.**
5. **원칙 위반 1회 = 전략 파괴**

## 📊 3가지 시장 상황 판정

매일 아침 이 한 가지만 확인하면 됩니다:

```
TQQQ 종가와 200일선의 관계는?
│
├─ 200일선 "아래"            → 🟦 하락 상황
├─ 200일선 위 & +5% "아래"    → 🟥 집중 투자 상황
└─ 200일선 +5% "위"          → 🟧 과열 상황
```

### 🟦 하락 상황 (TQQQ 종가 < 200일선)
- **메시지**: "시장 바닥을 예측하지 말고 현금으로 대피하라"
- **기존 보유자**: TQQQ·SPYM 전량 매도 → **SGOV 100%**
- **월급 등 신규 자금**: 모두 SGOV에 보관, 집중투자 신호 올 때까지 대기
- **실행 시점**: **당일 아침 애프터마켓** (지연 금지)

### 🟥 집중 투자 상황 (200일선 < TQQQ < 200일선 × 1.05)
- **메시지**: "상승 추세가 확인됐다. 수익을 극대화할 핵심 구간"
- **기존 보유자**: SGOV 전량 → **TQQQ 매수** (기존 SPYM은 그대로 유지 = 무게추)
- **월급/목돈 신규**: **TQQQ 매수**
- **실행 시점**: **다음 날 아침 애프터마켓** ("상승 시 하루 더 확인" 원칙)
- **매수 직후 필수**: 평균단가 대비 **-5% 시장가 스탑로스 주문 걸어두기**

### 🟧 과열 상황 (TQQQ > 200일선 × 1.05)
- **메시지**: "상승은 여전하지만 추격매수는 위험"
- **기존 TQQQ 보유**: **그대로 유지** (익절 원칙에 따라 부분 매도는 OK)
- **월급 등 신규 자금**: **SPYM 매수** (TQQQ 추격매수 금지)
- **실행 시점**: 다음 날 정규장 또는 애프터마켓

## 🕐 매매 체크리스트 (매일 아침 루틴)

### Step 1. 종가 확인 (1분)
- [ ] 증권사 앱에서 TQQQ 차트 열기
- [ ] 200일선(차트 설정에서 추가) 확인
- [ ] TQQQ 종가가 200일선 **위**인지 **아래**인지 확인
- [ ] +5% Envelope 위인지 아래인지 확인

### Step 2. 상황 판정 (30초)
| 어제 상태 | 오늘 상태 | 해야 할 일 |
|---|---|---|
| 현금(SGOV) 보유 중 | 200일선 아래 | ✅ 유지 |
| 현금(SGOV) 보유 중 | 200일선 **위** (신규 돌파) | ⏳ **내일 한 번 더 확인**, 그때도 위면 TQQQ 매수 |
| 현금(SGOV) 보유 중 | 200일선 위 (어제도 위였음) | 🟥 **오늘 TQQQ 매수** |
| TQQQ 보유 중 | 200일선 **아래** | 🟦 **오늘 즉시 전량 매도 → SGOV** |
| TQQQ 보유 중 | 200일선 위 (+5% 아래) | ✅ 유지, 월급은 TQQQ 추가매수 |
| TQQQ 보유 중 | 200일선 +5% 위 | ✅ TQQQ 유지, 월급은 **SPYM**으로 |

### Step 3. 주문 실행 (2분)
- [ ] 매도 신호면 → **즉시 아침 애프터마켓** 시장가 매도
- [ ] 매수 신호면 → **다음 날 아침 애프터마켓** 시장가 매수
- [ ] 매수 즉시 → **-5% 스탑로스** 주문 설정

### Step 4. 기록 (1분)
- [ ] 거래일, 매수/매도가, 수량, 사유(신호 발생) 기록
- [ ] 다음 체크일 달력에 표시

## 🛡 스탑로스 규칙 (TQQQ 전용)

| 항목 | 값 |
|---|---|
| 트리거 | 내 평균 매수 단가 대비 **-5% 이하** |
| 주문 유형 | 시장가 (Market) |
| 설정 시점 | TQQQ 매수 체결 직후 (앱 끄기 전) |
| 발동 시 조치 | TQQQ 매도 + **SPYM도 함께 매도** → SGOV 전환 |

> **왜?** -5% 스탑로스는 곧 "200일선 이탈 가능성"의 조기 경보이기 때문.
> 발동 다음 날 시장이 반등해도 감정 개입 금지. **원칙이 감정보다 먼저.**

## 💰 익절 규칙 (TQQQ 전용, 선택)

### 공식 원칙 (초보자 권장) — 단계별 분할 익절
내 평균 매수 단가 대비:

| 도달 수익률 | 매도 수량 | 누적 효과 |
|---|---|---|
| **+10%** | 보유량의 10% | 심리적 안정 |
| **+25%** | 남은 것의 10% | 원금 일부 확보 |
| **+50%** | 남은 것의 10% | 추가 안전 |
| **+100%, +200%, …** | 그때 남은 것의 **50%** | 복리 수익 확정 |

- 익절로 생긴 현금 → **SPYM에 재투자** (수익 보존)
- 고액이면 → 원금은 SPYM, 수익금은 SGOV 분리

### 최소 원칙 (단순함 선호자)
- **부분 익절 없음**, 오직 **200일선 이탈 시 전량 매도**

## 📅 연간/특수 상황 가이드

### 🎁 목표 금액 달성 (전세금, 내 집 마련)
- **200일선 규칙보다 우선**
- 목표 달성 즉시 **TQQQ·SPYM 전량 매도**
- 원금 → SPYM, 수익금 → SGOV로 분리 보관

### 💸 은퇴기 월 인출 (자체 배당)
- 월 0.5% (연 6%) = 자산 성장형
- 월 1% (연 12%) = 자산 가치 보존형
- **하락 상황**: SGOV 매도 → 인출
- **상승 상황**: ① SPYM 우선 매도, ② 부족 시 TQQQ 매도

### 🧾 양도소득세 납부 (매년 5월)
- 미리 현금 확보 X, 투자 중단 X
- 4월 말 ~ 5월 초, **상황별 인출 원칙**대로 필요 금액만 매도

## ⚠️ 절대 하지 말 것 (금기)

- ❌ **예측**: "곧 떨어질 것 같다" / "지금이 바닥인 것 같다"
- ❌ **뉴스 기반 매매**: FOMC, 고용지표, 실적 발표로 포지션 변경
- ❌ **손절 회피**: -5% 스탑로스 발동 후 재매수 ("감정 개입")
- ❌ **추격 매수**: 과열 구간에서 "FOMO"로 TQQQ 추가 매수
- ❌ **집중 깨기**: 집중투자 신호에서 금액을 쪼개서 분할 진입
- ❌ **원칙 변경**: 손절 몇 번 나면 "이번에는 다르다"며 기준 바꾸기

## 🧠 심리 관리

### "휩쏘가 너무 자주 일어난다" (실전 현실)
- 54년 백테스트 기준 **승률 33%**, 세 번 중 두 번은 손절
- 손절 중위값은 **-3%** — 감당할 수 있는 작은 손실
- **-3% 손실은 실패가 아니라 "저렴한 보험료"**
- 한 번의 +40%~+578% 거래가 모든 손실을 갚는다

### "손절 나자마자 시장이 올라간다" (가장 흔한 고통)
- 흔한 일이며, 이를 피할 방법은 없다
- **감정으로 재진입 X** — 다음 신호를 기다린다
- "아, 하루만 더 참을걸" 이라는 아쉬움은 정상, **따라가지만 말 것**

### "200일선이 의미 없어 보인다" (상승장 후반)
- 상승장이 길수록 "규칙 무용론"이 유혹한다
- 하지만 대형 하락은 **예고 없이** 온다
- 규칙을 지킨 사람만이 MDD -53%로 -99.96%를 피할 수 있다

| 주기 | 해야 할 일 |
|---|---|
| **매일 아침** | TQQQ 종가 vs 200일선 확인 → 상황 판정 → 필요시 주문 |
| **매주** | 거래 기록 간단 리뷰 (원칙 준수 여부 점검) |

| **연말** | 1년 거래 내역 회고, 원칙 위반 사례 점검 |

## 📱 증권사 앱 세팅 체크리스트 (삼성증권 기준)

- [ ] TQQQ 차트에 **200일 이동평균선** 추가 (설정 → 보조지표 → 주가이동평균 → 200일, 굵기 5px)
- [ ] TQQQ 차트에 **Envelope(200일, +5%)** 추가
- [ ] 차트 유형을 **라인(Line)** 으로 설정 (돌파 여부 깔끔하게 확인)
- [ ] TQQQ/SPYM/SGOV **관심종목 등록**
- [ ] **애프터마켓 거래 활성화** 확인
- [ ] TQQQ 매수 시 **자동 스탑로스(-5% 시장가)** 주문 프리셋 저장

## 🆘 예외 상황 대응

| 상황 | 대응 |
|---|---|
| **주말/공휴일에 신호 발생** | 다음 영업일 아침에 처리 |
| **종가가 200일선에 정확히 일치** | "위" 로 간주하지 않고 **다음날 재확인** |
| **장중 급등락 (intraday)** | 무시. **종가**만이 신호다 |
| **해외 출장/여행 중** | 모바일로 5분 체크만 하면 됨, 자동 스탑로스가 안전망 |
| **신호 놓침 (1~2일 지연)** | 놓친 그 날의 종가 기준으로 **지금이라도 실행** |
| **여러 신호 동시 발생** (드묾) | 우선순위: 매도 신호 > 매수 신호 |
